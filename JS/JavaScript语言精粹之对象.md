**JavaScript 语言精粹之对象**


----------


　　今天阅读的是《JavaScript语言精粹》的第三章“对象”，这本书比较薄，每一章也都不长，下面对书中提到的一些细节的知识点再进行总结：

   1、对象是属性的容器，其中每个属性都拥有名字和值。属性的名字可以是包括空字符串在内的任意字符串。属性值可以是除undefined 值之外的任何值。

   2、在对象字面量中，如果属性名是一个合法的JavaScript 标识符且不是保留字，并不强制要求用引号括住属性名。所以用引号括住“first-name”是必须的，但是否括住first_name是可选的。因此在设置对象的属性名时要注意是否按照JS语法对标识符的要求进行命名，否则一定要加上引号。

3、 在利用for in 对对象中的属性进行枚举时，要注意属性名出现的顺序与原有所写顺序不一定完全一致。如下所示：
　　

```
var obj = {
			2:"2",
			40:"40",
			15:"15"
		};
		for (var k in obj) {
			console.log(k);
		};//2 15 40
```
　　因此应尽量避免依赖对象属性顺序的代码。如果想顺序遍历一组数据，应使用数组并使用for语句遍历。
