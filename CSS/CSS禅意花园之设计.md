**CSS禅意花园之设计**
----------
 　　在《CSS禅意花园》这本书中，在第二章主要介绍了设计方面的一些技巧。一些技巧包括：
 　　1、将最主要的、起支配地位的元素用艺术线条或照片表示，或在悉心调整匹配区域的大小后再用线条环绕等；
 　　2、借喻（用简单的设计元素表示复杂的图像或概念）是一种强大的设计方法，可以为作品添加一些微妙的气氛并增加作品的深度；
 　　3、可以用光和影为作品添加真实感，或者让作品中的某个部分凸显出来，以体现层次感。例如让图像实现梯度效果、为按钮添加立体的斜边，给照片创建阴影等。如果使用Photoshop实现阴影效果，需确保选中了“Use Global Angle”单选框。这样即可让作品中各个阴影的效果保持一致。
 　　4、圆弧形经常与女性、阴柔关联起来，表示温暖、舒适、暧昧以及爱情等含义。很多以女性、社区、整体生活或浪漫传奇为主题的网站经常使用圆弧形来强调其主题。
 　　5、三角形会让人联想到男性、阳刚，用来表达诸如强壮、攻击以及运动感等含义。棱角的方向也会影响形状的含义-浏览者的目光通常会被吸引至处于支配地位的棱角所指向的方向。指向上方体现了向上的趋势以及攻击性，而指向下方则通常暗示出消极的意味，表示被动和无力。
 　　6、方形暗示了力量和根基，这很可能因为它们看上去足够的坚实稳重。方形的设计能给浏览者带来有序、逻辑、严密以及安全的感觉。
 　　7、空间感在心理和社会方面都有极为明显的含义。
 　　8、对于某些颜色，大多数人对其的理解和认同感都惊人的相似。这也是蓝色、棕色以及绿色被广泛应用于各种作品中的原因。
 　　9、对比度差别可以用来区分不同的元素。若对比度过低，则元素将失去彼此的界限并合并在一起，文字也变得难以辨认。若对比度过高，则将给浏览者带来压迫感，同样不会觉得舒适。
