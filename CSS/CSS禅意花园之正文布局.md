**CSS禅意花园之设计**
----------
　　对于《CSS禅意花园》的第三章正文布局，我只看了两三节，就有点看不下去了，感觉看这本书收获不是很大，不知是不是我有点急躁，有点想弃书了。。。。　　

　　第三章的第一节主要介绍了其中一个利用分栏布局的网页作品，作品是个单页网页，作者利用了绝对布局进行，但是在现在很多的网页布局中，应该不仅仅只用一个分栏布局就好了，所以我认为利用绝对定位太局限了，应该是不提倡的。另书中也提到了利用浮动进行布局，这个自不必多说。在第二节中提到了几种常规的水平居中的方式并介绍了固定布局和流式布局，这些也是比较常见的基本知识。因此打算先把这本书缓一缓，明天想看的是[The Subtle Magic Behind Why the Bootstrap 3 Grid Works](http://www.helloerik.com/the-subtle-magic-behind-why-the-bootstrap-3-grid-works),[Bootstrap3.0 栅格系统背后的精妙魔法(Bootstrap3.0的栅格布局系统实现原理)](http://www.cnblogs.com/willian/p/3558180.html?utm_source=tuicool&utm_medium=referral)
